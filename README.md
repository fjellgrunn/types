# @fjell/types

Core types and interfaces for the Fjell ecosystem. This package contains pure type definitions with zero runtime dependencies.

## Installation

```bash
npm install @fjell/types
```

## Usage

```typescript
import { Item, PriKey, ComKey } from '@fjell/types';

// Use types in your code
const myKey: PriKey = {
  pk: 'my-pk',
  sk: 'my-sk'
};
```
